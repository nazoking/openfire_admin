# OpenfireAdmin

Controll Openfire Admin by ruby. (by web scraping)
Openfire is xmpp server for internet chat.
for Opscode chef.

## Installation

Add this line to your application's Gemfile:

    gem 'openfire_admin'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install openfire_admin
